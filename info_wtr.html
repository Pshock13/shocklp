This simple add-on renames the workflow tabs so that they are more relevent to the content they display

As you can see in the images above, the first shows all the tabs have the same name even though they are each different pages with different information on them.
The second image shows the tabs have been renamed to an appropriate title.

Also, the tray count is added in the tab of the tray count page so you don't have to switch to that tab in order to see the number.

Note: If there are any tabs I missed that you'd like to see have a rename added, please feel free to let me know. -shocklp@
