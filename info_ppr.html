PPR Highlighter Info Page

By: Phillip Shockley [shocklp]
  • [Info]
  • [Changelog]

This simple add-on adds check boxes next to each item, highlighting the row when checked. This allows you to much more easily find the items you need. Items checked will persist through refreshes and even computer shutdown.

* 1.8 *
  • Negative highlight now flashes red
  • Red highlight for under 100% now gets removed when unchecking corresponding box
* 1.6 *
  • Percentages over 1,000% now highlight in green rather than red
* 1.0 - 1.5 *
  • Checkboxes on right side of table to select rows
  • Highlight color set to #8ff.
  • Positive percentages highlight set to #afa.
  • Negative percentages highlight set to #faa.
